# KNSI E-Xpert Blog

Repozytorium zawiera artykuły i aktualności związane z działalnością Koła Naukowego Systemów Informatycznych E-xpert.

## Struktura projektu

### Katalog `news/`

Katalog zawiera artykuły chronologicznie ponumerowane (od najstarszego):

- **001 warsztaty ionic.md** (2017-02-26) - Warsztaty z Ionic – projektowanie aplikacji mobilnych
- **002 konferencja zak.md** (2017-02-27) - Konferencja ŻAK – współorganizacja przez KNSI E-xpert
- **003 konkurs ais 2017.md** (2017-04-15) - Międzynarodowy Konkurs AIS Student Chapter 2017
- **004 spotkanie rekrutacyjne 2017.md** (2017-10-23) - Spotkanie rekrutacyjne KNSI E-xpert 2017/2018

## Format artykułów

Każdy artykuł zawiera metadane w nagłówku będącym komentarzem Markdown:
- `title` - tytuł artykułu
- `date` - data publikacji (YYYY-MM-DD)
- `author` - autor artykułu
- `description` - krótki opis
- `tags` - tagi kategoryzujące artykuł

## Konwencja nazewnictwa

Pliki w katalogu `news/` są nazywane według wzoru:
```
[numer] [krótki tytuł].md
```

gdzie:
- `[numer]` - trzycyfrowy numer porządkowy (001, 002, itd.) określający chronologię (najstarszy artykuł = 001)
- `[krótki tytuł]` - skrócona wersja tytułu artykułu z małych liter, ze spacjami
